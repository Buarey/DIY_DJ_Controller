# Security And Safety

This is a DIY hardware and firmware project. It is provided without warranty; builders are responsible for safe soldering, wiring, USB power use, enclosure assembly, and local electrical rules.

## Reporting Problems

If you find an issue that could damage hardware, create unsafe heat, short USB power, expose a computer to electrical faults, or mislead builders into unsafe wiring, please open an issue and mark it clearly as a safety concern.

For sensitive reports that should not be public immediately, contact the maintainer privately through GitHub before posting full details.

## Scope

Useful reports include:

- Wiring mistakes or ambiguous diagrams.
- PCB shorts, footprint mistakes, or pinout mismatches.
- Firmware behavior that can lock controls or send unsafe/unexpected MIDI floods.
- Documentation that could cause a bad build.

This project does not currently handle credentials, network services, or private user data.
